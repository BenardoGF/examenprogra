# CinePOO - Sistema de Gestión de Películas
Autores:
Garcia Fragoso Bernardo
Cisneros López Edwin
Ramos Gómez Daniel Ricardo


Este es un sistema simple de gestión de películas desarrollado con Flask y programación orientada a objetos.

## Características

- Gestión completa de películas (CRUD)
- Interfaz web responsive con Bootstrap
- Manejo de imágenes para las películas
- Programación orientada a objetos

## Requisitos

- Python 3.x
- Flask
- Bootstrap 5 (incluido vía CDN)
- Font Awesome (incluido vía CDN)

## Instalación

1. Clonar el repositorio
2. Instalar Flask:
   ```
   pip install flask
   ```
3. Ejecutar la aplicación:
   ```
   python app.py
   ```

## Estructura

- `app.py`: Aplicación principal Flask
- `pelicula.py`: Clase para manejar las películas
- `templates/`: Archivos HTML
- `static/img/`: Imágenes de las películas

## Uso

1. Ejecutar la aplicación
2. Abrir en el navegador: http://localhost:5000
3. Navegar por las diferentes opciones:
   - Ver lista de películas
   - Agregar nueva película
   - Editar película existente
   - Eliminar película
